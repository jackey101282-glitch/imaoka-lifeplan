## 0. 結論（先に一言）
4チームの結論が驚くほど一致しました。**「家計管理という“目的”の手前に、“使うのが楽しい・貼りたい・見せたい・育てたい”という感情のゴールを置く」**のが続く鍵。
そして Mietta の核（予定×お金）と最も相性がいいのは——
- **予定を“入れたくなる”** → カレンダーをデコれる（スタンプ）
- **続けたくなる** → ふたりの相棒キャラが育つ／月末に思い出が残る

**最初の一手はこの3つを推します（全チームで重複推薦・実装が軽い・既存資産を流用）。**

## 1. なぜ続くのか（リサーチ＝顧客インサイト）
女性が手帳デコ・シール・日記を“続けてしまう”心理：
- **収集欲／コンプリート衝動**（エンダウド・プログレス効果：最初から20%埋まってると完走率1.8倍。Nunes&Drèze 2006）
- **自己表現と所有感**（自分の色・字が残る。「育てるプロセス」が愛される）
- **小さな完了の快感**（チェック・スタンプ＝ドーパミン。Zeigarnik効果）
- **見返す快楽／ノスタルジア**（過去の記録が次の入力動機になる）

家計簿が**続かない死因**と、その裏返し：
| 死因 | 裏返しの設計原則 |
|---|---|
| 入力が面倒(34.5%) | 1タップ・数秒・選択式 |
| 成果が見えない | 入力するたびに“何かが育つ/貯まる”ビジュアル |
| 責められる感(赤字が怖い) | 罰でなく肯定。「先月比改善」を前面に |
| 自分好みにできない(離脱1位) | テーマ・アイコン・カテゴリのカスタム |
| ストリーク恐怖 | 連続より“累積の肯定”（休んでも消えない） |

**夫婦ならでは**：共有で「自分のため→ふたりのコミュニケーション」に昇格。妻の入力→夫のワンタップ→妻に感謝通知の“感謝ループ”が効く。記録が“ふたりの歴史帳”になると離脱しにくい。

→ **設計原則8か条**：①罰を消し肯定だけ ②入力を自己表現に ③最初から進んでる状態で始める ④小さな完了をかわいく祝う（毎回でなく“変動報酬”でたまにレア演出）⑤夫婦の感謝ループ ⑥記録を歴史帳に ⑦完成より“育てる” ⑧カスタムの余地。

## 3. 機能案（企画・全11／詳細は別途）
推し3つ＋有力どころ：
1. **🏆 カレンダーデコ（スタンプ）**【軽・無料＋有料拡張】予定にスタンプを貼れる。旅行=カメラ、デート=花、記念日=ケーキ。「今月のカレンダーかわいくしたい」で予定を早く入れる。旅行/外食スタンプに**予算タグ**を紐づけ＝予定×お金が自然に繋がる。SNS映え→口コミ。
2. **🏆 ふたりの相棒キャラが育つ（マスコット育成）**【中・無料＋有料スキン】予定入力＝エサ、家計入力＝水、ゴール達成＝進化。**罰なし**（休むと眠るだけ）。既存のコイン演出/動物ランクの延長で軽い。
3. **🏆 月末「ふたりの思い出アルバム」自動生成**【軽〜中・無料＋有料テーマ】達成ゴール＋完了予定＋動物ランクを1枚に。既存の月次レポート画像生成を流用。SNSシェアで拡散。歴史帳ロック。
- 予定→支出「キャプチャ」プロンプト【軽・無料】過ぎた予定に「いくらかかった？」→タップで支出登録。予定×お金の核を最小実装。
- 記念日カウントダウン＆お祝いカード【軽・無料】記念日7日前からホームに表示、当日に紙吹雪。プレゼント予算タグでゴール連動。
- 達成スタンプ帳（バッジ図鑑）／季節の着せ替え／ごほうびコイン→ショップ／ライフフェーズ着せ替えホーム／ポジティブ版“ふたりの連続記録” ほか。

## 4. 世界観（デザイン）
既存ブランド（Mietta＝和テイスト＆温かみ、テラコッタ/モスグリーン/クリーム、ラインアイコン、絵文字は使わない）を**壊さない**前提。
- **マスコット推し＝「コモリちゃん」**：甲羅がノートになった小さなカメ。アプリアイコン（ノート＋ハート）のDNAを継ぎ、動物ランクの“そこそこカメ”とは役割分離（あちらはスコア、こちらは語りかけ相棒）。予定を入れると甲羅が光る／記録ゼロだと眠る。バリアント3種（通常・点灯・眠り）から低リスクで始める。
- **デコ/スタンプの質感**：「インクが滲んだ和紙スタンプ」。1スタンプ最大3色・必ず1色はブランド色から。鮮色（青ピンク）は紙吹雪専用。角丸大きめ。季節モチーフ（桜/金魚/銀杏/雪）はSVGで自動切替。
- **ごほうび演出の3段階**：Lv1通常（現状のコイン＋紙吹雪）→Lv2 streak節目（キャラが一言）→Lv3 ゴール達成（紙吹雪増量＋キャラが花丸）。`prefers-reduced-motion`配下に必ず入れる。
- **NG**：絵文字復活／原色ベタ／キャラの喋りすぎ／ネオングラデ／全部アニメ化／キャラをランクアイコンに混ぜる。
