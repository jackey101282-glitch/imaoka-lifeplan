# アイデアボード：将来CFを「毎日見たい・払いたい」へ（2026-07-02 総動員ブレスト）

> 社長指示：ゼロベースで数を出す（できる/やる度外視）＋社長がしそうな突っ込みを先回り。
> 部隊：competitor / trend / insight / product / design-ui / bizdev ＋ Codex ＋ NotebookLM。総アイデア100超→以下に収束整理。

## 通底する「払う理由」＝3つ（全チャネル一致）
1. **未来込みの「今日いくら使っていいか」** — 家計簿は"今月の残額"。Miettaは住宅・教育・働き方・老後まで織り込んだ"使っていい額"を出せる。毎日開く核。
2. **夫婦で"決める"インフラ** — 合意ライン／一緒に見た証／自由費ガードレール。単身家計簿に作れない。NotebookLM/Luminix「Couples Gap」が実際の有料転換要因。
3. **「良くなった実感」＝解約されない** — 効いた行動の履歴／元取り／「何もしなかったらとの差分」。

## 地雷（やってはいけない・全チャネル一致）
- 不安を煽る赤字/破綻/老後不足演出（怖いと一度は開くが毎日見ない・世界観に反する）
- 入力を増やして精度を追う（課金前に離脱）→ プリセット・仮置き・あとで修正
- ただのグラフ追加（谷/推移/比較だけは「見た」で終わる）→ 今日の判断・夫婦の合意・安心に"変換"する

## 収束した本命アイデア（複数AIが独立提案＝高シグナル）
| テーマ | 何を見せる | 効き（毎日/課金） | 社長の突っ込み→返し |
|---|---|---|---|
| **今日使っていい額** | 未来(住宅/教育/働き方/老後)込みの"今日の自由予算" | 毎日開く核 | 「無料家計簿でよくない?」→無料は今月残額、Miettaは未来込み＋夫婦2人合算 |
| **what-ifスライダー** | 貯蓄額/時期/働き方を動かすと谷と月額が即再計算 | 行動できる数字・有料の核 | 「入力多い」→スライダー1本、既定値を動かすだけ |
| **カレンダー日別残高予測** | カレンダーに各日の予測残高。給料日/引落が一目 | 毎日見る動機・Mietta独自(カレンダー統合) | 「毎日見たい?」→まさにこれ。既存カレンダーに上乗せ |
| **夫婦の合意ライン/一緒に見た証** | 相談不要額の自動提案・「2人とも見ました」記録・自由費ガードレール | 夫婦ならでは・課金理由 | 「監視っぽい」→"一緒に見た記念"の前向き設計 |
| **効いた行動履歴/元取り/差分** | 見直しで安心残高が◯万増・「何もしなかったら」との差 | 解約防止・良くなった実感 | 「いやらしい」→月次に控えめ表示 |
| **今日の一手/今週の小さな修正** | 「外食あと2回減で谷が◯万改善」等 谷を消す最短行動 | 見て終わりを防ぐ | 「当たり前」→効果を"何年後の谷が◯万改善"で根拠化 |

| **NISA複利ワクワク/目標資産逆算** | 「今から月1万→◯年後◯円」/「◯歳で◯円に必要な月額」/自由になれる年齢 | 増える実感 | 「元本割れ」→保守/標準/楽観の3本併記 |

| **穏やかベンチマーク** | 同年代夫婦の目安との差(順位・下位は出さない) | 位置がわかる安心 | 「煽り/世界観」→公的統計・"参考"トーン・否定語禁止 |
| **上品な可視化** | 資産の川(Sankey和風)/ヒートマップ/純資産1本線/スクラブ | 説得力・体験・SNS映え(金額は隠せる) | 「複雑/世界観」→帯3本に単純化・和配色・タップで深掘り |
| **静かなAI一言** | 「先月より外食が◯円多いようです」等の気づき(説教でなく) | 毎日違う・開く理由 | 「胡散臭い/コスト」→定型文でも可・断定しない |

## 本命TOP4（課金の決め手）
1. **今日使っていい額**（未来込み・夫婦合算）＝家計簿の弱点に答える毎日の核
2. **what-ifスライダー**＝"選択の結果"を触って見る＝行動が変わる＝ProjectionLabの本命
3. **夫婦の合意ライン/一緒に見る**＝夫婦特化の必然（Couples Gapが課金要因）
4. **効いた行動履歴/元取り**＝払い続ける理由・解約防止

## 社長がこの提案に言いそうなこと → 先回り
- 「結局どれも予測でしょ」→ 本命は全て**予測を"今日の行動・夫婦の合意・良くなった実感"に変換**したもの。ただのグラフは地雷として除外済み。
- 「無料でできる」→ 差別化は**未来込み×夫婦2人合算×カレンダー統合×価値観すり合わせ**の組み合わせ。この束は競合ゼロ（既往リサーチ）。
- 「入力が増える」→ 既定値・プリセット・スライダーで**入力ゼロ〜1本**。
- 「世界観に合う?」→ 演出は和・静か・線アイコン・否定語禁止で統一。
- 「毎日見る?」→ 今日使っていい額＋待受ダッシュボード＋カレンダー残高が毎日の入口。
