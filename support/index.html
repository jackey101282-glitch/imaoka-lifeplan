よくある質問

データは安全ですか？

銀行口座やクレジットカードとは連携せず、口座番号・カード番号・明細などの重い情報は預かりません。データはスペース（世帯）ごとに分離して保管され、ご本人と招待した相手だけが見られます。通信は暗号化（HTTPS）しています。

アカウントやデータを削除したい

アプリ内の「設定 → アカウント削除」から、アカウントと関連データを削除できます。

パートナーと一緒に使うには？

「設定 → パートナーを招待」から招待リンク（または招待コード）を送り、相手がそれを開いて参加すると、同じスペースを共有できます。ひとりで使い始めて、あとから招待することもできます。

使い方が分からない

「設定 → はじめてガイドをもう一度見る」から、最初の使い方を確認できます。

サブスクリプションの解約方法は？

iPhoneの「設定 → ご自身のApple ID → サブスクリプション」からMiettaを選び、解約できます。解約後も、契約期間の終了までは引き続きご利用いただけます。

退会した場合、共有していたデータはどうなりますか？

スペースに自分だけの場合はデータも削除されます。パートナーが残っている場合の取り扱いは、削除時にアプリ内でご案内します。
